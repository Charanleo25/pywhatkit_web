PyWhatKit

A Python library with various helpful features. It is an easy to use library which does not requires you to do some additional setup. Currently it has about 200k+ downloads. The source code of this library can be found on GitHub.


Started From Here:

>pip install pywhatkit

Sending WhatsApp Messages
